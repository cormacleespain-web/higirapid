# AI Product Team — Export Package

This zip contains the full AI Product Team so you can use it on another device.

## Contents

- **.cursor/agents/** — 13 subagent definitions (.md)
- **.cursor/skills/** — All role skills: SKILL.md, templates.md, guidelines, guardrails (.md)
- **.cursor/rules/** — Team coordination, documentation standards, output quality (.mdc)
- **workflows/** — Feature development, sprint planning, design review, release readiness (.md)
- **docs/** — Shared docs structure and status (.md)

## How to use on another device

1. Unzip into your project or Cursor workspace root (keep the same folder structure).
2. Ensure **.cursor/** (with the leading dot) is present at the top level after unzip.
3. Cursor uses .cursor/agents/, .cursor/skills/, and .cursor/rules/ from the workspace.
